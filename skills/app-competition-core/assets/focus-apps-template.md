# Focus Apps

## Rules

- 这里只记录“当前重点关注”的 app，不记录所有候选。
- `watch_type` 使用：`target` / `direct` / `potential` / `substitute`
- `status` 使用：`active` / `watching` / `paused` / `archived`
- discovery 产出的 `Track` 项自动写入这里；`Watch` 项留在项目总控文件里。
