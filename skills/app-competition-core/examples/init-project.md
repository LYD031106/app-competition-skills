# Example: Initialize A Project

用户请求：

```text
帮我给这个项目建立 app 竞品分析目录，需要 app-compete.md 和 focus-apps.md。
```

预期动作：
- 创建 `docs/app-competition/`
- 创建 `app-compete.md`
- 创建 `focus-apps.md`
- 创建 `reports/`、`snapshots/`、`evidence/`
- 明确提醒用户补充 `target_app` 与 `default_market`

不该做的事：
- 不要擅自假设默认市场
- 不要先写任何竞品报告
